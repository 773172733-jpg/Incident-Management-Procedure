# 事件树 Codex 接管包使用方法

## 放置文件

把本压缩包解压后：

1. 将 `AGENTS.md` 复制到微信小程序项目根目录。
2. 将整个 `docs` 文件夹复制到项目根目录。
3. 将整个 `design` 文件夹复制到项目根目录。
4. `prompts` 文件夹可以保留在项目里，也可以放在桌面备用。

最终结构至少应包含：

```text
你的微信小程序项目/
├─ AGENTS.md
├─ docs/
│  ├─ 事件树微信小程序_单人版架构与团队预留_V1.0.md
│  ├─ UI_SPEC.md
│  └─ CODEX_HANDOFF.md
├─ design/
│  └─ 事件树_UI设计稿参考.png
└─ 原有小程序代码...
```

## 第一次让Codex接管

在Codex中打开“整个微信小程序项目文件夹”，不要只打开某个代码文件。

复制：
`prompts/01_首次接管_只审计修复阶段0.txt`

第一次只让Codex审计并修复阶段0。

## 阶段0通过后

复制：
`prompts/02_阶段1_首页与大事件.txt`

不要一句话让Codex做完整小程序。每一阶段在微信开发者工具测试通过后，再继续下一阶段。
